# Завод Навоза 2.0 — Автономный генератор экзистенциальной поэзии

## Описание

Модуль генерирует стихи в стиле «классика $\times$ абсурд» на тему навоза и бетона, используя LLM API.

## Запуск

Для запуска модуля из корня проекта используйте команду:

```bash
python3 -m src.main
```

## Требования

- Python 3.10+
- Заполненный файл `.env` с ключами доступа к API.

## Структура

- `src/generator.py`: Работа с API LLM, промпты и парсинг.
- `src/recorder.py`: Атомарная запись в JSON с защитой от повреждений и изоляцией путей.
- `src/notifier.py`: Система уведомлений (сейчас — заглушка в stdout).
- `src/main.py`: Точка входа и оркестрация компонентов.
